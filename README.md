# LiCs-Bitter-Coil
Design files and instructions for Bitter coils built in the LiCs experiment in Chin Lab @ UChicago. 

The key authors of the files in this repository are Henry Ando, Leon Gold, and Cheng Chin. This repository is intended as an informational resource for the cold atom community. At the moment, it contains the following:

1. A PDF assembly guide written by Leon Gold, which outlines the key steps in our assembly process.
2. A folder containing all the CAD files for the coil we built. We designed it in Autodesk Fusion 360, so the 3D files are in that format. I have also included minimal 2D drawings, which are mostly for reference.
3. A draft of a manuscript describing the more general improvements we plan to make.
